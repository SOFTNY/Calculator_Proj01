## 📌Main Function
#### switch-case문 연산처리
```javascript
switch()
{
  case '+':
    result = a + b;
    break;
  case '-':
    result = a - b;
    break;
  case 'x':
    result = a * b;
    break;
  case '/':
    result = a / b;
    break;
  default:
    return;
}
```
